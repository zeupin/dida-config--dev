# Dida\Config 组件库

`Dida\Config` 是一个 App 配置组件，可以方便地对 App 的配置项进行各种高级设置。它是 [宙品科技](http://zeupin.com) 开源的 [Dida 框架](http://dida.zeupin.com) 的一个功能组件。
